# Introduction to the Zettelkasten Method

## Tesis o aporte principal

Un Zettelkasten util necesita identificadores unicos, notas atomicas, referencias claras y enlaces con contexto explicito para construir una red real de conocimiento.

## Notas de lectura

- Cada zettel debe tener un identificador unico.
- La nota debe contener una sola unidad de pensamiento o conocimiento.
- Las referencias deben dejar claro de donde proviene la idea.
- Los enlaces sin contexto no generan conocimiento util.
- Las structure notes actuan como mapas de navegacion y no como simples listas.
- El sistema funciona mejor cuando se apoya en texto plano y convenciones duraderas.

## Citas textuales
