# El sistema importa mas que la meta

Esta idea fue inspirada por [[Habitos Atomicos]].

No sirve de mucho tener una meta por si sola. Tanto perdedores como ganadores pueden tener metas parecidas; la diferencia real esta en el sistema que sostiene las acciones diarias y vuelve posible llegar a ese resultado.
