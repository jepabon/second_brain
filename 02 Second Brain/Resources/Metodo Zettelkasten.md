# Metodo Zettelkasten

## Resumen

Nota estructural para agrupar recursos y lecturas sobre Zettelkasten como metodo de gestion de conocimiento, enlazando conceptos, lecturas base y futuros desarrollos del second brain local.

## Ideas clave

- Zettelkasten no es solo tomar notas, sino enlazar ideas con contexto.
- Las notas estructurales sirven como puntos de entrada y navegacion.
- El sistema local en markdown es una base mas durable para este tipo de metodo que una base de datos cargada de vistas.

## Uso en este vault

- Funciona como nota puente entre la metodologia GTD y la evolucion de `02 Second Brain`.
- Enlaza el marco conceptual con la lectura base [[Introduction to the Zettelkasten Method]].
- Puede crecer luego en un mapa de contenido o structure note real.
