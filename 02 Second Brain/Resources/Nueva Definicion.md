# Nueva Definicion

## Resumen

Documento de criterio para un sistema personal que unifique captura, GTD, media tracking, second brain, rutinas, finanzas y registros personales.

## Ideas clave

- Toda la informacion debe entrar por una bandeja de entrada.
- La informacion se clasifica entre accionable y no accionable.
- Las acciones se dividen en calendario, siguientes acciones, delegadas y proyectos.
- Lo no accionable incluye papelera, algun dia y consulta.
- El sistema debe soportar planificacion diaria, semanal y mensual.
- Debe integrar media tracking, contactos, mascotas, finanzas y zettelkasten.

## Fragmentos migrados

- Se agrega todo tipo de informacion, sin distincion de ningun tipo.
- Permite clasificar la informacion en diferentes tipos.
- Permite hacer una planificacion para el dia, la semana y al mes.
- Permite llevar un registro de libros, mangas, animes, peliculas, series, etc.
- Permite llevar un registro tipo zettelkasten para aprender nuevos pensamientos.

## Nota

Este recurso sirve como puente conceptual entre tu sistema antiguo en Notion y la nueva arquitectura local en Obsidian.
